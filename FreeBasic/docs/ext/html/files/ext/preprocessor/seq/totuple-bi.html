fbextPP_SeqToTuple

# define fbextPP_SeqToTuple(seq) (FBEXT_PP_SEQ_ENUM(seq))

Converts a sequence into a tuple.

Parameters

seqthe sequence to convert

Returns

Returns the tuple representation of the sequence.

Description

Given a sequence (a)(b)(c)(d), this macro expands to (a, b, c, d).
