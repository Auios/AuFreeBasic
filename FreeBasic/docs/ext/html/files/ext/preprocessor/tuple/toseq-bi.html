fbextPP_TupleToSeq

# define fbextPP_TupleToSeq(size,
tuple) fbextPP_TupleToSeq__##size##tuple

converts the tuple tuple of size size into a sequence.

For example, if size is `3` and tuple is `(a,b,c)`, then this macro expands to `(a)(b)(c)`.

Parameters

sizeis the size of the tuple.
tupleis the tuple to convert.
