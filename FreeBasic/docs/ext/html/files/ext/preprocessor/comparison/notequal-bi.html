Macros

fbextPP_NotEqual

# define fbextPP_NotEqual(a,
b) fbextPP_Not(fbextPP_Equal(a, b))

expands to `1` if a and b do not compare equal, otherwise it expands to `0`.  a and b must be integer literals in the range [0, 255], inclusive.

FBEXT_PP_NOTEQUAL

# define FBEXT_PP_NOTEQUAL(a,
b) fbextPP_NotEqual(a, b)

is deprecated, use fbextPP_NotEqual instead.
