t,
f) fbextPP_Iif(fbextPP_Bool(c), t, f)

Expands one of two texts depending on a numeric condition.

Parameters

cA number.  This number is converted to a boolean value.  If this conversion is not necessary, use fbextPP_Iif instead.
tthe text to expand if c is non-zero.
fthe text to expand if c is zero.

Returns

One of the two texts.
