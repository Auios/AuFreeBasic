Macros

fbextPP_TupleReverse

# define fbextPP_TupleReverse(size,
tuple) fbextPP_TupleReverse__##size tuple

reverses the order of elements stored in the tuple tuple of size size.

For example, if size is `3` and tuple is `(a,b,c)`, then this macro expands to `(c,b,a)`.

Parameters

sizeis the count of elements in the tuple.
tupleis the tuple to reverse.
