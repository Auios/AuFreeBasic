tokenize

declare function tokenize(byval x as zstring ptr =  0,
byref tkns as const string,  
byval lastfound as ubyte ptr =  0,
byval eattkns as bool =  true) as string

Splits a string into tokens, similiar to C’s strtok.

Parameters

xzstring ptr to string to tokenize, should only be passed at first call.
tknsconst string containing the tokens to match on.
lastfoundubyte ptr, will return the character code of the last found token (0 for EOL)

Returns

String portion delimited by the first found token (skipping multiple at the beginning if necessary.

Example

#include once "ext/strings/substr.bi"

var x = "This is a string, and this is the same one."
dim as ubyte what = 0

var t = ext.strings.tokenize(x, " ,.", @what)

while t <> ""
  print using !"&  \"&\""; t; chr(what)
  t = ext.strings.tokenize(, " ,.", @what)
wend
